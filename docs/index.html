Linear A deciphered as an extinct Semitic language

added 2026-06-26 · next review 2026-07-03 (in 7d) · source: link

Review log

The claim

Tom Di Mino, a self-taught AI engineer and amateur linguist in the Hudson

Valley, claims to have deciphered Linear A — the Bronze Age Minoan script

undeciphered for over a century. He proposes it maps to an **extinct Semitic

language** predating Biblical Hebrew, Arabic, and Aramaic.

Specifics:

  • Proposed readings for ~40 script signs, including 13 previously unknown.
  • A lexicon of 408 Linear A terms translated into English.
  • Identifies a verb root "nawaya" ("to dwell"); claims his solution also

resolves open problems in the (already-deciphered) Linear B.

  • Built a suite of Python scripts using Claude Code to query and

cross-reference the digitized Linear A corpus for systematic hypothesis

testing. Has a draft 9-page manuscript.

Reported via AI Clambake and

discussed on Hacker News,

June 2026.

Why it needs watching

  • Classic outsider profile: amateur linguist + AI tooling, no academic

affiliation. Linear A decipherment claims are a magnet for cranks — the test

is whether epigraphers/Semiticists endorse it.

  • Author's claimed validation ("under review by linguistics experts at Rutgers

and Cambridge") is unverified and not yet peer-reviewed.

  • Expert skepticism already raised: the corpus is tiny and fragmentary; we

can't be sure all "Linear A" texts share one language; if it were Semitic,

Greek substrate words should already show Semitic etymologies and largely

don't (per commentary on HN).
